# Privacy Policy for PinScheduler Bot

**Last Updated: June 8, 2026**

This Privacy Policy explains how the PinScheduler automation tool ("the Application") handles data. The Application is a private, personal-use automation tool designed to streamline content scheduling from Telegram to a specific, authorized Pinterest Business profile.

## 1. Data Collection and Usage
The Application collects and processes only the information explicitly provided by the authorized operator via the Telegram Bot interface. This data includes:
* **Product URLs:** Affiliate or retail product links provided for scheduling.
* **Media Assets:** Product images extracted from the provided links to create Pinterest Pins.
* **Content Copy:** Post titles, descriptions, and metadata generated by the Gemini API based on the product inputs.

## 2. Third-Party Data Processing
To fulfill its scheduling functions, the Application securely passes data to the following platforms:
* **Telegram Bot API:** To receive user instructions and links.
* **Google Gemini API:** To analyze product data and generate localized marketing descriptions.
* **Google Sheets API / Web App:** To log scheduled posts into a private queue.
* **Pinterest API (v5):** To programmatically publish the final Pins to the operator's authorized boards.

## 3. Data Storage and Security
* **No Public Databases:** The Application does not store user data on public or multi-tenant servers. All scheduling metadata is stored within the operator's private Google Sheet.
* **Credential Protection:** API keys and access tokens (Telegram, Gemini, Pinterest) are stored securely as environment variables on the host environment and are never hardcoded or exposed publicly.

## 4. Operational Scope
The Application is strictly for private, individual content management. It does not offer public sign-ups, multi-user accounts, or commercially process external user data.
